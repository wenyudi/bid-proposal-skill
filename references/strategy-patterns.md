# 一页纸策略骨架

本文件服务 Task 1 的"想"阶段：提供一页纸启发式骨架、标型先验和叙事选型。章节结构来自标书，项目事实来自当前输入。Task 1 据此发散多个候选主张，再收敛一版。

## 共同骨架
一版能贯穿全案的策略应能连续回答：
1. 客户表面要什么，真正难在什么张力？
2. 我们发现了什么非行业常识的事实或因果？
3. 因此全案只做哪一个战略选择，又明确不做什么？
4. 评委翻完后能复述哪一句话？
5. 这个选择如何推导创意表达、执行机制和证明方式？
6. 每章为这条推导链贡献哪一个独有判断，并向后交出什么？
7. 与它最接近的可行替代是什么，两者真正争夺哪一个取舍？
8. 哪一个客户可见成果最可能让评委立即理解并记住这条主张？

候选阶段从 outcome、efficiency、risk、visibility、experience、asset、contrarian 多镜头探索；每个候选讲清因果与适用条件。

## 六类常见标型（标型先验）
| 标型 | 适用信号 | 评委重点 | 优先补的证据/交付 | 典型客户可见成果 |
|:---|:---|:---|:---|:---|
| 品牌传播 / IMC | 年度传播、形象升级、整合战役 | 策略闭环、大概念、跨渠道性价比 | 受众/品牌依据、创意机制、渠道协同、版权舆情 | 内容样例 + 渠道编排 + 传播时序 |
| 活动 / 事件营销 | 发布会、节庆、路演、展会、赛事 | 体验亮点、现场落地、安全应急 | run-of-show、场地/人员容量、审批、彩排、fallback | 分钟级流程 + 体验节点 + 应急卡 |
| 数字营销 / 新媒体 | 代运营、社媒矩阵、内容/投放增长 | 持续产能、数据闭环、平台合规 | 基线、栏目产能、排班、平台规则、数据源与频率 | 内容日历 + 样帖 + 看板 |
| 政务宣传 / 城市形象 | 政策解读、城市/文旅、主题宣传 | 导向、权威性、公共价值、舆情安全 | 政策原文、公开城市事实、受众依据、审校机制 | 叙事样例 + 内容矩阵 + 审校路径 |
| 影视 / 内容制作 | 宣传片、纪录片、TVC、短视频 | 立意/质感、样片、里程碑、版权 | 样片角色、主创可用性、脚本分镜、素材授权、审改验收 | 含时间码/画面/声音的分镜 + 视觉系统 |
| 媒介采购 / 投放 | 户外/线上/电视采购与执行 | 资源真实性、价格、监播、结案 | 刊例/授权、库存点位、折扣依据、覆盖测算、监播补量 | 计算样例 + 排期分配 + 监播证据样式 |

先读本标要求和角色依据；表中角色/成果可判不适用，也可加入未列内容。章节服从评分骨架，不为补齐标型新增孤儿章。成果示例只提示"评委需要看到什么"，仍要依本项目真实边界收敛，不照表编造。

## 政府与公共机构
"好"首先是公共任务的正当性、连续性、可审计性与风险妥帖，然后才是新奇。优先寻找：政策目标与公众真实使用的落差；协同/审核/舆情/运营之间的张力；"被看见"与"被理解、参与、采用"的差别；一次性声量与长期公共资产的取舍。常用推导："已有基础进入下一阶段，需要从 A 转向 B"。把审批、基层或采购人的管理负担计入价值判断。不用商业焦虑贬低现状，不以声量替代公共结果。

## 商业客户
"好"首先是生意问题、客户行为与经济机制的洞察，然后才是传播形式。优先寻找：增长目标与真实决策摩擦的矛盾；认知—购买—渠道之间的断点；同质供给下客户为何不选/迟选/低价选；短期转化与长期品牌资产可被管理的取舍。核心主张必须说明改变哪种客户行为、凭什么改变、怎样验证。只写"提升声量、扩大影响、形成闭环"不构成商业策略。

## 混合场景
国企市场化业务、文旅经营或公共品牌可能同时存在两套标准。明确本标以 `government_public` 还是 `commercial` 为主、另一套作约束，让核心选择保持清晰；不能平均成"品牌 + 社会效益双提升"套话。

## 叙事选型
按标型从 `narratives.json` 的四个 key 自动选一个主叙事：
| mode | 优先信号 | 主要风险 |
|:---|:---|:---|
| `logic` | 技术/服务方案重、暗标、机制复杂 | 变成枯燥说明书 |
| `story` | 城市/文旅/公益/品牌、创意契合度高、需讲标 | 虚构场景盖过评分响应 |
| `vision` | 多年框架、长期代运营、明确战略语境 | 愿景无本合同期交付 |
| `evidence` | 效果/投放/增长、量化与报价确定性强 | 数据越权、目标冒充保证 |

选择顺序：命中信号最多 → 最高权重评分维度定夺。主叙事全案唯一；叙事只决定表达顺序与温度，不裁剪评分项、不加强承诺。报价、合规、资质段落固定用事实/逻辑，不套叙事。

## 真实赢标案抽取模板
获得 3–5 份已确认赢标案后，逐份只抽：项目类型/客户语境/竞争强度/已知中标原因；客户张力与依据；核心洞察、主张、记忆句；洞察→策略→创意→执行→证明的推导；每章独有贡献；哪些表达只有该投标人能成立；评委/客户实际复述或表扬的内容；哪些属于合规门/价格/关系变量（归因时与文案质量分开）。跨至少三案重复且有反馈支持的结构可升为先验；单次中标只是待验证候选。
